## Description

[NestJS](https://github.com/nestjs/nest) framework based REST API for executing JSON rules.

Simple wrapper for [json-rules-engine](https://github.com/CacheControl/json-rules-engine)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
